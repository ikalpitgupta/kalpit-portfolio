# Kalpit Gupta - Personal Portfolio

A modern, responsive personal portfolio website built with React and Tailwind CSS, showcasing the professional profile of Kalpit Gupta, a B.Tech student specializing in Computer Science and Engineering (Bioinformatics) at VIT.

## Features

- **Modern Design**: Clean, professional, and tech-oriented design
- **Responsive**: Fully responsive across all devices
- **Dark/Light Mode**: Toggle between dark and light themes
- **Smooth Animations**: Subtle animations and hover effects
- **Fast Loading**: Optimized for performance
- **SEO Friendly**: Proper meta tags and semantic HTML

## Sections

- **Hero Section**: Introduction with contact information and social links
- **About**: Personal bio and key strengths
- **Education**: Academic background and achievements
- **Technical Skills**: Programming languages, technologies, and proficiency levels
- **Work Experience**: Professional internship experiences
- **Projects**: Featured projects with detailed descriptions
- **Extracurricular**: Leadership roles and activities
- **Certifications & Achievements**: Professional certifications and awards
- **Contact**: Contact information and social media links

## Technologies Used

- **Frontend**: React 18, Tailwind CSS
- **Fonts**: Inter, Poppins (Google Fonts)
- **Icons**: Heroicons (SVG)
- **Build Tool**: Create React App
- **Styling**: Tailwind CSS with custom configurations

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd kalpit-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.

## Customization

### Colors
The color scheme can be customized in `tailwind.config.js`:
- Primary colors (blue shades)
- Accent colors (orange and gold)

### Content
Update personal information in the respective component files:
- `src/components/Hero.js` - Contact information
- `src/components/About.js` - Personal bio
- `src/components/Education.js` - Educational background
- `src/components/Experience.js` - Work experience
- `src/components/Projects.js` - Project details

### Fonts
Fonts are loaded from Google Fonts in `public/index.html`. Current fonts:
- Inter (primary)
- Poppins (headings)
